Allows you to incorporate interactive range sliders into Elementor forms.

== Description ==

[youtube https://www.youtube.com/watch?v=hOmCB-bVraA]

**DEMO**: <https://demo.add-ons.org/demo-range-slider-field/>

Elementor Forms Range Slider Fields is a valuable feature that allows you to incorporate interactive range sliders into Elementor forms. This feature enables users to select a value within a specific range by sliding a handle along a graphical slider.

== Features ==
- **Styling Options**: Customize the appearance of the slider with color and style options to match your website's design.
- **Supports Multiple Data Formats**: Display and collect a variety of data types, including integers, decimals, and custom formats.
- **Responsive Design**: Fully responsive and optimized for mobile and desktop views, ensuring a seamless experience on all devices.
- **Easy Integration**: Simple to add and configure within Elementor's form builder, no coding required.
- **Customizable Range**: Define minimum, maximum, and default values for the slider to suit your form's needs.
- **Conditional Logic Support**: Use with conditional fields to show or hide elements based on the slider's value for more dynamic forms.
- **Multi-Language Support**: Integrates with multilingual plugins like WPML, making it easy to use on international websites.

Document: <https://add-ons.org/document-wpfoms-repeater-fields-2/>

== Installation ==
**Normal installation**

1. Download the repeater-field-for-wpforms.zip file to your computer.
2. Unzip the file.
3. Upload the `repeater-field-for-wpforms` directory to your `/wp-content/plugins/` directory.
4. Activate the plugin through the 'Plugins' menu in WordPress.
5. Document: <https://add-ons.org/document-wpfoms-repeater-fields-2/>
